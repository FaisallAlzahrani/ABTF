## Getting Started

### Accessing the Module
1. Log in to the application with your credentials
2. From the home screen, tap on the "Planning & Inspection" tile
3. You will be directed to the Planning & Inspection screen with two tabs:
   - Planning (default tab)
   - Inspection

### Understanding the Dashboard
The Planning dashboard provides:
- A list of all work orders
- Status filters to sort work orders
- A floating action button to create new work orders

## Creating Work Orders

### Step-by-Step Process
1. From the Planning dashboard, tap the "+" button in the bottom right corner
2. Fill in the required information:
   - **Work Order Number**: Enter a unique identifier for the work order
   - **Project Name**: Enter the name of the project
   - **Item Description**: Enter a description of the item being inspected
   - **Quantity**: Enter the quantity of items
   - **Due Date**: Select a due date for the work order
3. Upload the ROUTECARD PDF file:
   - Tap "Select RouteCard PDF"
   - Browse to and select the PDF file
   - The system will display the number of pages detected in the PDF
4. Tap "Create Work Order" to save

### PDF Processing
- After uploading the PDF, the system will automatically process it
- Each page of the PDF will be split into an individual inspection sheet
- This process may take a few moments depending on the size of the PDF
- You'll see a progress indicator during processing

## Managing Work Orders

### Viewing Work Orders
1. From the Planning dashboard, you'll see a list of all work orders
2. Each work order card displays:
   - Work Order Number
   - Project Name
   - Item Description
   - Quantity
   - Status
   - Created Date
   - Due Date
   - Number of inspection sheets

### Filtering Work Orders
1. Use the filter chips at the top of the screen to filter by status:
   - All
   - Draft
   - Pending
   - In Progress
   - Completed
   - Rejected
2. Tap the filter icon in the app bar for additional filtering options

### Viewing Work Order Details
1. Tap on a work order card to view its details
2. The details screen shows:
   - Work order information
   - A list of all inspection sheets
   - Action buttons

### Updating Work Order Status
1. From the work order details screen, tap the menu icon in the top right
2. Select the desired status:
   - Mark as Draft
   - Mark as Pending
   - Mark as In Progress
   - Mark as Completed
   - Mark as Rejected

## Working with Inspection Sheets

### Viewing Inspection Sheets
1. From the work order details screen, tap on an inspection sheet
2. The inspection sheet viewer will open, showing the PDF page

### Adding Inspection Data
1. In the inspection sheet viewer, tap the edit icon in the top right
2. Fill in the inspection form:
   - Select a status (Pending, Passed, Failed, Waived)
   - Add any remarks
3. Tap "Save" to save the inspection data

### Viewing Inspection History
1. In the inspection sheet viewer, when not in edit mode, you can see:
   - Current status
   - Remarks
   - Inspector information
   - Inspection date

## Troubleshooting

### PDF Upload Issues
- Ensure the PDF file is not corrupted
- Try uploading a smaller PDF file if you're experiencing issues
- Check your internet connection

### PDF Processing Issues
- If PDF processing fails, try uploading the PDF again
- Ensure the PDF is not password-protected
- Contact IT support if issues persist

### Work Order Creation Issues
- Ensure all required fields are filled out
- Check your internet connection
- Try again later if the system is experiencing high load

### Inspection Sheet Viewing Issues
- If an inspection sheet fails to load, try refreshing the page
- Check your internet connection
- Contact IT support if issues persist

For additional support, please contact the IT department.
