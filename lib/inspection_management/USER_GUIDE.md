## Getting Started

### Accessing the Module
1. Log in to the application with your credentials
2. From the home screen, tap on the "Inspection" tile
3. You will be directed to the Inspection Management dashboard

### Understanding the Dashboard
The dashboard provides quick access to:
- View Work Orders
- My Inspections (for inspectors)
- Pending Approvals (for administrators)
- Reports (coming soon)

## For Planning Department

### Creating a New Work Order
1. From the Inspection Management dashboard, tap "View Work Orders"
2. Tap the "+" button in the bottom right corner
3. Fill in the required information:
   - Work Order Number
   - Project Name
   - Description
   - Due Date
4. Assign inspectors by selecting them from the list
5. Upload the ROUTECARD PDF file by tapping "Select RouteCard PDF"
6. Tap "Create Work Order" to save

### Monitoring Work Orders
1. From the Work Orders list, you can see all work orders and their statuses
2. Use the filter options to sort by status, date, or assigned inspectors
3. Tap on any work order to view its details
4. From the details screen, you can:
   - View the work order status
   - See assigned inspectors
   - Check inspection progress
   - View the RouteCard PDF

## For Inspectors

### Finding Assigned Work Orders
1. From the Inspection Management dashboard, tap "My Inspections"
2. You will see a list of work orders assigned to you
3. Work orders are color-coded by status:
   - Orange: Pending
   - Blue: In Progress
   - Green: Completed
   - Red: Rejected

### Performing Inspections
1. Tap on a work order to view its details
2. Tap "View RouteCard" to open the PDF viewer
3. Navigate through the PDF pages (each page represents a piece mark)
4. Use the tools at the bottom of the screen:
   - Tap the "Inspection Form" icon to enter inspection data
   - Tap the "Annotation" icon to add annotations to the PDF

### Using the Annotation Tools
1. With annotation mode active, you can:
   - Draw on the PDF using the pen tool
   - Highlight areas using the highlighter tool
   - Erase annotations using the eraser tool
   - Add text annotations
   - Add approval stamps
2. Adjust the color and stroke width using the controls at the bottom
3. Tap "Save Annotation" when finished

### Filling Inspection Forms
1. With the inspection form open, enter values for each inspection point
2. Select the result (Pass/Fail/N/A) for each point
3. Add new inspection points if needed by tapping "Add Inspection Point"
4. Add any remarks in the remarks field
5. Sign the inspection by tapping "Sign" and drawing your signature
6. Tap "Save Inspection Data" when complete

## For Administrators

### Reviewing Work Orders
1. From the Inspection Management dashboard, access all work orders
2. Filter by status to find work orders requiring review
3. Tap on a work order to view its details and inspection results

### Approving or Rejecting Inspections
1. From the work order details screen, review the inspection data
2. View the RouteCard with annotations
3. Check inspector signatures and inspection points
4. Use the menu in the top right to:
   - Mark as Pending (if more work is needed)
   - Mark as In Progress
   - Mark as Completed (if approved)
   - Mark as Rejected (if not approved)

## Troubleshooting

### Offline Mode
- The app can work offline, allowing you to view and annotate PDFs
- Inspection data will be saved locally and synced when you're back online
- A notification will appear when data is synced

### PDF Loading Issues
- If a PDF fails to load, try the following:
  1. Check your internet connection
  2. Close and reopen the PDF viewer
  3. Restart the application
  4. Contact IT support if the issue persists

### Data Not Saving
- Ensure you tap the "Save" button after making changes
- Check your internet connection
- If working offline, ensure you sync when back online

### Access Issues
- If you cannot access certain features, verify your user role with your administrator
- Different roles have different permissions within the module

For additional support, please contact the IT department.
